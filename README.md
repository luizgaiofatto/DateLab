# DateLab

## Link
https://date-lab.vercel.app

DateLab é uma aplicação web que gera ideias de encontros de forma aleatória e permite compartilhar o resultado diretamente pelo WhatsApp.

## Funcionalidades

* Geração aleatória de dates
* Interface simples e intuitiva
* Animação ao sortear o encontro
* Envio do date via WhatsApp
* Campo para inserir o número do parceiro(a)
* Salvamento automático do número com localStorage
* Aplicação instalável (PWA)

## Tecnologias

* HTML5
* CSS3
* JavaScript (Vanilla)
* PWA (Manifest + Service Worker)

## Como usar

1. Clique em "Sortear" para gerar um date
2. Insira o número do parceiro(a) com DDD
3. Clique em "Confirmar"
4. O WhatsApp será aberto com a mensagem pronta para envio

## Persistência

O número inserido é salvo automaticamente no navegador, evitando a necessidade de digitar novamente.

## Instalação (PWA)

A aplicação pode ser instalada como um app:

1. Acesse o site pelo navegador
2. Abra o menu do navegador
3. Clique em "Adicionar à tela inicial" ou "Instalar app"

## Objetivo

Projeto desenvolvido para uso pessoal com foco em prática de:

* Manipulação de DOM
* Lógica de programação
* Experiência do usuário (UX)
* Integração com WhatsApp
* Desenvolvimento de Progressive Web Apps
